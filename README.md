# 텍스트 이미지 변환기
## 개요
입력한 텍스트를 선택한 스타일을 입혀서 텍스트로 변환해주는 서비스입니다.
## 탄생 배경
Github README에는 텍스트에 색상을 입히는 행위가 금지되어 있습니다.  
보안의 일종같은데, 가끔 README를 예쁘게 꾸미고 싶어지는건 어쩔 수 없죠.  
그래서 색상 텍스트를 이미지로 대체하면 어떨까 싶어서 만든 서비스입니다.
